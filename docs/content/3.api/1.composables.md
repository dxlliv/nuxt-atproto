Types: [`src/types/index.d.ts`](https://github.com/dxlliv/nuxt-atproto/blob/main/src/types/index.d.ts). `OAuthSession` is re-exported from [`@atproto/oauth-client-browser`](https://www.npmjs.com/package/@atproto/oauth-client-browser) — use `session.sub` as the account DID.

## `useAtprotoSession()`

### Returns

| Name | Type | Server | Description |
|------|------|--------|-------------|
| `session` | `Ref<OAuthSession \| undefined>` | always `undefined` | Current OAuth session |
| `isLogged` | `ComputedRef<boolean>` | always `false` | Whether a session exists |
| `status` | `Ref<AtprotoSessionStatus>` | always `'anonymous'` | `'initializing'`, `'authenticated'`, or `'anonymous'` on the client |

---

## `useAtprotoAuth()`

### Methods

| Method | Signature | Server | Throws |
|--------|-----------|--------|--------|
| `signIn` | `(serviceEndpoint?, options?) => Promise<void>` | no-op | no — logs errors |
| `signInWithHandle` | `(handle: string, options?) => Promise<void>` | no-op | no — logs errors |
| `signOut` | `() => Promise<void>` | no-op | no — logs revoke errors |
| `restore` | `(did: string) => Promise<OAuthSession>` | throws | yes — `'useAtprotoAuth is only available on the client'` |

---

## `useAtprotoAgent(scope, fetch?)`

### Parameters

| Param | Type | Description |
|-------|------|-------------|
| `scope` | `'authenticated' \| 'public' \| string` | Agent type and service URL |
| `fetch` | `typeof fetch` (optional) | Custom fetch for `AtpAgent` |

### Returns

`Agent` when `scope === 'authenticated'`, otherwise `AtpAgent`.

| Context | Behavior |
|---------|----------|
| Server | throws `'useAtprotoAgent is only available on the client'` |
| Client, `authenticated`, no session | throws `'Not authenticated'` |

---

## `useAtproto()` (deprecated)

Combines session + auth. `signInWithHandle()` without arguments uses `window.prompt`. Use `useAtprotoSession` + `useAtprotoAuth`.

---

## `useAgent(service, fetch?)` (deprecated)

| `service` | Maps to |
|-----------|---------|
| `'private'` | `useAtprotoAgent('authenticated')` |
| `'public'` | `useAtprotoAgent('public')` |
| other string | `useAtprotoAgent(url)` |

Conceptual usage: [Authentication](/guide/authentication), [Agents](/guide/agents).
