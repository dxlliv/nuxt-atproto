## Compatibility

| | Supported |
|---|-----------|
| **nuxt-atproto** | `0.1.0` (current) |
| **Nuxt** | 4.x |
| **Node.js** | 20 LTS or newer |
| **Browsers** | Modern evergreen browsers with OAuth redirect support |

Peer dependency: [`@atproto/api`](https://www.npmjs.com/package/@atproto/api) and [`@atproto/oauth-client-browser`](https://www.npmjs.com/package/@atproto/oauth-client-browser) are bundled as module dependencies — your app typically uses them via composables, not as direct installs.

## Release history

See [GitHub Releases](https://github.com/dxlliv/nuxt-atproto/releases) for version notes.

This project uses [changelogen](https://github.com/unjs/changelogen) for release notes. After upgrading, run `nuxt prepare` and check [Migration](/advanced/migration) if you used legacy `useAtproto()` or `useAgent()`.

## API surface (0.1.0+)

| Recommended | Deprecated |
|-------------|------------|
| `useAtprotoSession()` | `useAtproto()` |
| `useAtprotoAuth()` | — |
| `useAtprotoAgent(scope)` | `useAgent(service)` |

Feed, post, and other Bluesky APIs are provided by [`@atproto/api`](https://www.npmjs.com/package/@atproto/api) — not documented in this module's reference.
