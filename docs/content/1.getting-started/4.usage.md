## Minimal login page

```vue [app.vue]
<script setup lang="ts">
const { isLogged, session, status } = useAtprotoSession()
const { signIn, signInWithHandle, signOut } = useAtprotoAuth()

const handle = ref('')
</script>

<template>
  <ClientOnly>
    <p v-if="status === 'initializing'">
      Loading session…
    </p>

    <div v-else-if="!isLogged">
      <button type="button" @click="signIn()">
        Sign in with ATProto
      </button>

      <input v-model="handle" placeholder="handle.example.com">
      <button
        type="button"
        :disabled="!handle"
        @click="signInWithHandle(handle)"
      >
        Sign in with handle
      </button>
    </div>

    <div v-else>
      <p>Signed in as {{ session?.sub }}</p>
      <button type="button" @click="signOut()">
        Sign out
      </button>
    </div>
  </ClientOnly>
</template>
```

::callout{type="info"}
Use `<ClientOnly>` (or `ssr: false` on the route) so login UI and OAuth state only run in the browser.
::

## Fetch data with an agent

```vue [pages/feed.vue]
<script setup lang="ts">
const { isLogged, status } = useAtprotoSession()

const timeline = ref<unknown>(null)
const error = ref<string | null>(null)

watch(
  () => [status.value, isLogged.value] as const,
  async ([currentStatus, loggedIn]) => {
    if (currentStatus === 'initializing' || !loggedIn) {
      timeline.value = null
      return
    }

    error.value = null

    try {
      const agent = useAtprotoAgent('authenticated')
      const { data } = await agent.getTimeline()
      timeline.value = data
    }
    catch (err) {
      error.value = err instanceof Error ? err.message : 'Failed to load timeline'
    }
  },
  { immediate: true },
)
</script>
```

::callout{type="warning"}
Prefer `watch` over `watchEffect(async …)` for agent calls — async effects can overlap when dependencies change quickly.
::

## React to session lifecycle

Register hooks in a client plugin or component that runs on the client:

```ts [plugins/atproto-hooks.client.ts]
export default defineNuxtPlugin(() => {
  const { hook } = useNuxtApp()

  hook('atproto:sessionCreated', async (did) => {
    console.log('New session:', did)
  })

  hook('atproto:sessionRestored', async (did) => {
    console.log('Restored session:', did)
  })

  hook('atproto:sessionDeleted', (did) => {
    console.log('Session removed:', did)
  })
})
```
