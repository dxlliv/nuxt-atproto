## Requirements

- **Node.js** 20 LTS or newer (matches Nuxt 4 requirements)
- [Nuxt](https://nuxt.com) **4.x** (module tested with Nuxt 4)
- ATProto must know the **URL where users open your app** (e.g. `https://myapp.com`) so it can redirect back after login — set this in [Configuration](/getting-started/configuration)

## Install the package

::steps
### Add the dependency

```bash
pnpm add nuxt-atproto
```

```bash
npm install nuxt-atproto
```

```bash
yarn add nuxt-atproto
```

### Register the module

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['nuxt-atproto'],
})
```

### Configure OAuth metadata

Before signing in works in production, set `atproto.oauth.clientMetadata` — see [Configuration](/getting-started/configuration).

### Run your app

```bash
pnpm dev
```

Composables are auto-imported; no manual imports required. Run `nuxt prepare` (or `pnpm dev`, which runs it) so `$atproto` types are generated.
::
